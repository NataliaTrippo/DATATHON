# DATATHON
Este projeto tem como objetivo analisar indicadores educacionais de alunos entre 2020 e 2022, incluindo desempenho acadêmico (IDA), engajamento (IEG), aspectos psicossociais (IPS), psicopedagógicos (IPP), adequação ao nível (IAN) e índice geral (INDE), a fim de identificar padrões de evolução, relações entre variáveis e fatores que influenciam o desenvolvimento dos estudantes. A análise contempla a evolução dos indicadores ao longo do tempo, a coerência entre autoavaliação e desempenho, o impacto de fatores psicossociais, bem como a efetividade do programa educacional nas diferentes fases (PEDRA). Além disso, foi desenvolvido um modelo preditivo de Machine Learning baseado em regressão logística para estimar a probabilidade de risco de defasagem dos alunos, utilizando múltiplos indicadores como variáveis explicativas. Os resultados são apresentados por meio de visualizações e de uma aplicação interativa em Streamlit, permitindo a exploração dos dados e a identificação precoce de alunos em situação de risco, contribuindo para tomadas de decisão mais assertivas no contexto educacional.
